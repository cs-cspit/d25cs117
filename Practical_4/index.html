GATE Q&A - HTML, CSS, JavaScript

HTML Q1: Which HTML tag is used to create a hyperlink?
Answer: <a> tag (Anchor tag). Example: <a href="https://example.com">Visit Example</a>
HTML Q2: In HTML5, which element represents self-contained content with a heading?
Answer: <article> element.
HTML Q3 (MCQ): Which attribute specifies an alternate text for an image?
Answer: B) alt
CSS Q4: Which CSS property changes the text color of an element?
Answer: color property. Example: p { color: red; }
CSS Q5: Difference between relative and absolute positioning?
Relative → positioned from its normal position. Absolute → positioned relative to the nearest positioned ancestor.
CSS Q6 (MCQ): Which property controls space between border and content?
Answer: B padding
JS Q7: Output of console.log(2 + "2")?
Answer: "22" (string concatenation).
JS Q8: Difference between == and === in JavaScript?
== compares values after type conversion. === compares values without type conversion.
JS Q9 (MCQ): Which of the following is NOT a JS data type? A) Number B) Boolean C) Character D) String
Answer: C) Character
